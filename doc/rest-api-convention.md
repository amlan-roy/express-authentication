# REST API convention

| CRUD Action                  | HTTP Method | Endpoint                |
| ---------------------------- | ----------- | ----------------------- |
| `Auth Route`                 |             |                         |
| Register the user            | POST        | `/api/auth/signup`      |
| Login the user               | POST        | `/api/auth/login`       |
| Get current user information | GET         | `/api/auth/currentUser` |
